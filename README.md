### Before you run the main.py
#### 1. Clone the Repo 
```
git clone https://github.com/kdavid001/H_I_com
```

#### 2. Go to the file location and initialize git
```
cd path/to/your/project/H_I_com
git init
git status 
```
>To confirm if you are connected to the remote repo: git remote -v

#### 3. Install the Packages(Flask)
```
pip install -r requirement.txt
```

##### 4. Run the app
```
python main.py
```
